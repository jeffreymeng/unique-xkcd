By default, this extension only keeps track of comics read while it is active. However, browser history can be loaded to avoid revisiting comics read before installation or on mobile:
Additionally, this extension can export and import data about viewed comics. This is useful when changing browsers or Google accounts. To export, click export and copy the resulting number. To import, paste the number from export and click import.
Type "Permanently Clear Data" (and click the button) to clear all history stored by this extension. We recommend you export your data before doing this, as this action is irreversible.
